# Sync Guardian Design System (Permanent)

These tokens MUST be used strictly for all UI components. No variations are permitted unless explicitly requested.

## 1. Colors
| Token | Hex Code | Usage |
| :--- | :--- | :--- |
| **Surface (Background)** | `#fff8f0` | Main screen background, status bar, navigation bar. |
| **Primary (Forest)** | `#486730` | Hero text, buttons, active states, progress indicators. |
| **Primary Container** | `#87a96b` | Gradient end-point, light surfaces. |
| **Secondary (Terracotta)** | `#9f402d` | Alert labels (e.g., "MORNING FLOW"), small accents. |
| **On-Surface** | `#1b1d0e` | Default dark text. |
| **On-Surface-Variant** | `#43483d` | Description text, secondary labels. |
| **Surface-Container** | `#efefd7` | Card backgrounds (Level 1). |
| **Surface-Container-Highest** | `#e4e4cc` | Floating elements, high-contrast cards. |
| **White (Pure)** | `#ffffff` | Pure highlights. |

## 2. Typography (Editorial)
- **Headlines**: `Plus Jakarta Sans` (Bold/SemiBold). 
- **Body**: `Manrope` (Regular/Medium).
- **Italics**: Use for emphasis in hero titles (e.g., *"digital sanctuary"*).

## 3. Shapes
- **Radius XL**: `32px` (Cards, Large Buttons).
- **Radius Full**: `9999px` (Pill buttons).
- **No Borders**: 1px dividers are forbidden. Use space or tonal shifts.

## 4. Components
- **Glassmorphism**: `surface_variant` at 60% opacity with 20px-32px background blur.
